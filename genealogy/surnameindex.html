The Surname Index

Read this to get started

This page contains alphabetical indexes for the family trees I've built and included on this site: Hannah's, Matt's and my own. Each tree has their own index below, and you can minimize the trees you're not interested in with the arrows on the right side of each one.

I've put some additional features in here to make navigation a little easier - each surname has a trail of links leading to a primary branch of that tree. Each of the surnames in the Vest-Horton-Scranton tree link to Vest, Horton, Scranton, or Jobe, etc.

Each of the entries of the index will (when I finally add them all) link directly to that ancestor's individual profile pages. Some other links in the profiles, like spouses, link to that ancestor's own entry in the index.

A star indicates that the entry is a direct ancestor of the subject of that tree. (Calvin, Hannah, or Matt.)
